GeoLead ⇄ GeoGuessr Bridge – Privacy Policy

Last updated: {{ last_updated }}

This browser extension (“Extension”) is provided by GeoLead (“we”, “us”). Its only purpose is to help you transfer your own GeoGuessr daily game results into your GeoLead account so that you don’t need to enter scores manually.

1. What the extension does

The Extension works in three simple steps:

  1. On geoguessr.com, it reads the game result data that is already available in the page (for example from GeoGuessr’s own JavaScript data) after you play a game.
  2. It stores a small summary of your latest game in the browser’s local extension storage.
  3. On the GeoLead submit page (geolead.fun), when you click “Import today’s game results”, it sends that summary to the GeoLead backend so that a score entry can be created or updated for you.

The Extension does not run on any websites other than geoguessr.com and geolead.fun.

2. Data we access

From GeoGuessr pages we may access the following data for the game you just played:

  • GeoGuessr profile information – such as your nickname, internal user ID and (if provided by GeoGuessr) email address.
  • Game metadata – such as game or daily quiz ID, game mode and date/time played.
  • Game results – scores per round, total score, round distances and coordinates for your guess and the correct location.

This data is read only for your own game and only for the purpose described here.

3. How we use the data

The Extension uses this data only to:

  • store your latest game summary in chrome.storage.local in your browser so it can be transferred between GeoGuessr and GeoLead; and
  • send the summary to the GeoLead backend when you click “Import today’s game results”, so that GeoLead can create or update your score entry and display your scores, distances and maps.

We do not use this data for advertising, profiling, or analytics that are unrelated to GeoLead.

4. Data sharing

  • The Extension sends your game summary only to the GeoLead service that you use (for example https://www.geolead.fun/).
  • We do not sell or transfer your data to third parties.
  • We do not use your data to determine creditworthiness or for lending.

5. Data retention

  • In the browser, the game summary is stored in the Extension’s local storage and is overwritten the next time you play a game and the Extension captures new results.
  • On the GeoLead side, your imported scores are stored only as long as needed to provide your leaderboard and history features. Retention is governed by GeoLead’s main site privacy policy.

6. Permissions

The Extension uses the following permissions:

  • storage – to temporarily store the latest game summary between geoguessr.com and geolead.fun.
  • Host permissions:
    • https://www.geoguessr.com/* – to read your own game data from GeoGuessr daily game pages.
    • https://www.geolead.fun/* – to inject the import button on the submit page and send your game result to GeoLead.

The Extension does not access your browsing activity outside these domains.
